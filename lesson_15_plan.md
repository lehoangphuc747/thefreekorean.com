
# Kế hoạch Siêu Cấp: Bổ Sung Biểu Hiện Từ Vựng Bài 15 (Phân Biệt 2 Format)

Mục tiêu: Phân biệt rõ ràng giữa TỪ GỐC (phân tích từ nguyên) và BIỂU HIỆN (phân tích cấu trúc).

## Format Giải Thích

### A. Từ Gốc (In Đậm) - Phân Tích Từ Nguyên

**Format Hán Việt:**
```
[Từ]: [Hán Việt].
[Chữ 1] ([Hán 1]): [giải thích nghĩa]
[Chữ 2] ([Hán 2]): [giải thích nghĩa]
→ Nghĩa: [tổng hợp]
```

**Ví dụ:**
```
도착: Đáo trước.
도 (Đáo): đến
착 (Trước): dính vào/tới nơi
→ Nghĩa: Đến nơi, tới đích.
```

**Format Từ Mượn:**
```
Từ mượn tiếng [Anh/Pháp/Việt] "[gốc]".
```

**Format Động Từ Thuần Hàn:**
```
Động từ thuần Hàn.
```

**Format Động Từ Ghép:**
```
[Phần 1]: nghĩa + [Phần 2]: nghĩa
→ Nghĩa: [tổng hợp]
```

### B. Biểu Hiện (Không In Đậm) - Phân Tích Cấu Trúc

**Lưu ý:** Chỉ dùng format này khi là BIỂU HIỆN (cụm từ/cấu trúc), không phải từ gốc.

```
[Động từ]: nghĩa
Cấu trúc: ...
Cấu trúc này có nghĩa là: ...
Trong câu ... có nghĩa là: ...
```

## ⚠️ Lưu Ý Quan Trọng

**Phân biệt rõ 2 loại:**

1. **Từ Gốc** (Hán Việt/Từ mượn/Động từ thuần Hàn):
   - Phân tích **TỪ NGUYÊN** (từng chữ Hán, nguồn gốc)
   - Ví dụ: **도착하다** → phân tích 도 (Đáo): đến, 착 (Trước): dính vào/tới nơi

2. **Biểu Hiện** (cụm từ/cấu trúc):
   - Phân tích **CẤU TRÚC** (cách dùng, ngữ pháp)
   - Ví dụ: **공항에 도착하다** → phân tích cấu trúc N에 도착하다

**KHÔNG BAO GIỜ** dùng format phân tích cấu trúc cho từ gốc, và **KHÔNG BAO GIỜ** dùng format phân tích từ nguyên cho biểu hiện.
