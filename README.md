# The Free Korean

Dự án website học tiếng Hàn miễn phí, sử dụng Astro framework.

## Tài liệu chính

- [TOPIK Vocabulary](topik_vocabulary.md) - Tổng hợp từ vựng TOPIK toàn diện
- [TOPIK Guide](TOPIK.md) - Hướng dẫn về TOPIK
- [Grammar Guide](grammar-guide.md) - Hướng dẫn ngữ pháp
- [Grammar List](GRAMMAR.md) - Danh sách ngữ pháp
- [Practice Structure](PRACTICE_STRUCTURE.md) - Cấu trúc luyện tập
- [Data Entry README](README_DATA_ENTRY.md) - Hướng dẫn nhập liệu

## Cấu trúc dự án

- `src/` - Mã nguồn chính
- `content/` - Nội dung (grammar, topik)
- `docs/` - Tài liệu bổ sung
- `public/` - Tài nguyên tĩnh

## Cách chạy

```bash
npm install
npm run dev
```
