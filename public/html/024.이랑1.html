1. 의미와 용법

여러 사물이나 사람을 연결함을 나타내는 조사.
(명사에 붙어) 사물이나 사람을 같은 자격으로 이어 주는 뜻을 나타낸다. 앞뒤 명사 모두를 가리킬 때 사용한다.
* 주로 구어(말하기)에서 많이 사용합니다.

2. 예문

담화 예문 문장 예문
가: 교실에 무엇이 있어요?
나: 책상이랑 의자가 있어요.
가: 편의점에서 무엇을 샀어요?
나: 빵이랑 라면을 샀어요.
가: 앤디 씨는 생일 선물을 받았어요?
나: 네, 꽃이랑 책을 받았어요.
가: 식당에서 누구를 만났어요?
나: 앤디랑 수지를 만났어요.
  • 마이클이랑 앤디는 제 친구예요.
  • 저는 수학이랑 컴퓨터를 배워요.
  • 저는 수박이랑 바나나를 좋아해요.
  • 동생은 밥이랑 김치를 먹어요.
  • 오늘이랑 내일은 수업이 없어요.
  • 언니랑 오빠가 중국으로 여행을 가요.
  • 철수랑 영희가 학교에 가요.
  • 왕밍은 바지랑 가방을 샀어요.
참고:
1. 순서를 바꾸어 써도 의미가 같습니다. (책상이랑 의자 = 의자랑 책상)
2. 셋 이상을 나열해도 됩니다. (옷이랑 구두랑 가방이랑 사요)
3. 격 조사(이/가, 을/를)는 맨 뒤에 오는 명사 뒤에 붙습니다.

3. 형태 정보

명사의 끝음절 받침 유무에 따라 '이랑' 또는 '랑'을 씁니다.

조건 형태 예시
받침 O (자음) 이랑 가방 + 이랑 → 가방이랑
연필 + 이랑 → 연필이랑
책 + 이랑 → 책이랑
받침 X (모음) 랑 컴퓨터 + 랑 → 컴퓨터랑
학교 + 랑 → 학교랑
의자 + 랑 → 의자랑

4. 제약 정보

① 문장과 문장을 연결할 때는 사용할 수 없습니다.

(X) 철수는 영어를 공부해요 랑 앤디는 한국어를 공부해요.

(O) 철수는 영어를 공부해요. 그리고 앤디는 한국어를 공부해요.

* 단, 두 문장의 구성이 같고 주어나 목적어만 다를 때는 합쳐서 표현할 수 있습니다.

철수는 영어를 공부해요. + 철수는 한국어를 공부해요.
→ 철수는 영어랑 한국어를 공부해요.
