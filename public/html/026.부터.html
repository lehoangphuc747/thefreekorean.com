1. 의미와 용법

어떤 동작이나 상태의 시작점을 나타내는 조사입니다.
크게 두 가지 의미로 사용됩니다.

시간 동작/상태의 시작: 어떤 동작이나 상태가 시작되는 시간을 나타냅니다.

순서 우선 순위: 여러 가지 일 중 순서나 서열상 제일 먼저 할 일을 나타냅니다.

2. 예문

담화 예문 문장 예문
시간
가: 선생님, 수업은 언제 시작해요?
나: 수업은 9시부터 시작해요.
시간
가: 흐엉 씨, 여름방학이 언제예요?
나: 여름방학은 6월부터 8월까지예요.
순서
가: 앤디 씨, 집에 가면 제일 먼저 무엇을 할 거예요?
나: 저는 잠부터 잘 거예요.
순서
가: 맛있는 음식이 많아서 뭐 부터 먹어야 할지 모르겠어요.
나: 가까이에 있는 것부터 차례로 드세요.
[시간의 시작]
  • 저는 오늘부터 도서관에서 공부할 거예요.
  • 흐엉 씨가 아침부터 삼겹살을 먹어요.
  • 현우 씨가 내일부터 다음 주까지 아르바이트를 해요.
  • 수지 씨는 그 약을 먹고부터 몸이 좋아졌어요.

[순서/우선]
  • 저는 아침에 신문부터 읽어요.
  • 철수부터 청소를 하기 시작했다.
  • 저는 다음부터 혼자 여행 가지 않을 거예요.

3. 형태 정보

명사의 끝음절 받침 유무에 관계없이 항상 '부터'를 씁니다.

조건 형태 예시
받침 O 부터 오늘 + 부터 → 오늘부터
동생 + 부터 → 동생부터
아침 + 부터 → 아침부터
받침 X 어제 + 부터 → 어제부터
언니 + 부터 → 언니부터
식사 + 부터 → 식사부터

4. 문장 구성 정보

'부터' ~ '까지'

어떤 행위나 상태의 시작(부터)과 끝(까지)을 나타내는 범위 표현으로 자주 쓰입니다.

  • 저는 9시부터 12시까지 한국어를 배웁니다.
  • 시험은 월요일부터 목요일까지입니다.
  • 철수는 아침부터 저녁까지 일을 합니다.
