# SEO Checklist cho các trang TOPIK

## Công cụ kiểm tra SEO miễn phí

### 1. Google Search Console
- **URL**: https://search.google.com/search-console
- **Chức năng**: 
  - Kiểm tra index status
  - Xem lỗi crawl
  - Kiểm tra performance
  - Submit sitemap

### 2. Google Rich Results Test
- **URL**: https://search.google.com/test/rich-results
- **Chức năng**: Kiểm tra structured data (JSON-LD)

### 3. PageSpeed Insights
- **URL**: https://pagespeed.web.dev/
- **Chức năng**: Kiểm tra tốc độ và Core Web Vitals

### 4. Schema Markup Validator
- **URL**: https://validator.schema.org/
- **Chức năng**: Kiểm tra structured data chính xác

### 5. Meta Tags Preview
- **URL**: https://metatags.io/
- **Chức năng**: Preview Open Graph và Twitter Cards

### 6. Screaming Frog SEO Spider (Free - 500 URLs)
- **URL**: https://www.screamingfrog.co.uk/seo-spider/
- **Chức nách**: Crawl và phân tích SEO

## Cải thiện SEO

### 1. Thêm nội dung mô tả chi tiết
- Thêm section "Giới thiệu" với 200-300 từ
- Mô tả nội dung đề thi
- Hướng dẫn sử dụng tài liệu

### 2. Thêm FAQ section
- Câu hỏi thường gặp về TOPIK
- Schema.org FAQPage structured data

### 3. Thêm Breadcrumb
- Schema.org BreadcrumbList
- Navigation rõ ràng

### 4. Tối ưu hình ảnh
- Alt text mô tả
- Compress images
- WebP format

### 5. Internal Linking
- Link đến các kỳ thi TOPIK khác
- Link đến các bài viết liên quan
- Link đến trang chủ

## Kiểm tra nhanh

1. **View Source** trang TOPIK và kiểm tra:
   - Title tag
   - Meta description
   - Structured data (JSON-LD)
   - Open Graph tags

2. **Google Search Console**:
   - Submit URL mới
   - Kiểm tra coverage
   - Monitor performance

3. **Rich Results Test**:
   - Paste URL vào tool
   - Kiểm tra structured data valid

## Lưu ý

- Cập nhật sitemap.xml khi thêm trang mới
- Kiểm tra duplicate content
- Đảm bảo mobile-friendly
- Tốc độ tải trang < 3 giây
- Core Web Vitals tốt
