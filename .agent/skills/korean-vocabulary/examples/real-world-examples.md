# Ví Dụ Thực Tế - Từ Vựng Hàn

Đây là các mẫu đã test thực tế, rút ra từ quá trình tạo file `THTH2-bai-01.md`.

## 1. Nhóm Từ Gốc + Biểu Hiện (Đúng Cấu Trúc)

```markdown
| 넣다 | | bỏ vào, cho vào | Động từ | Danh từ thuần Hàn.<br>Nghĩa: Hành động đặt vật phẩm vào bên trong một không gian nào đó. | 가방에 핸드폰을 **넣었어요**. | Tôi đã **cho** điện thoại vào túi. | | | 꺼내다 |
| 가방에 넣다 | | bỏ vào túi | Biểu hiện | **가방**: Túi. **넣다**: Bỏ vào.<br><br>Cấu trúc: **N1에 N2을/를 넣다**.<br>Cấu trúc này có nghĩa là: **Bỏ vật phẩm N2 vào không gian N1.**<br><br>Trong câu **가방에 넣다** có nghĩa là: **Bỏ (cái gì đó) vào túi.** | 책을 **가방에 넣었어요**. | Tôi đã **bỏ sách vào túi**. | | | 가방에서 꺼내다 |
| 가방에 휴대폰을 넣다 | | bỏ điện thoại vào túi | Biểu hiện | Kết hợp đầy đủ của **가방에 넣다** với **휴대폰** (điện thoại).<br><br>Trong câu này có nghĩa là: **Bỏ điện thoại vào túi.** | **가방에 휴대폰을 넣고** 나갔어요. | Tôi **bỏ điện thoại vào túi** rồi ra ngoài. | | | |
| 커피에 설탕을 넣다 | | cho đường vào cà phê | Biểu hiện | **커피**: Cà phê. **설탕**: Đường. **넣다**: Cho vào.<br><br>Kết hợp đầy đủ: **Cho đường vào cà phê.**<br><br>Trong câu này thể hiện hành động **pha chế** đồ uống. | **커피에 설탕을 두 스푼 넣었어요**. | Tôi đã **cho hai thìa đường vào cà phê**. | | | |
```

**Note:** Các biểu hiện xếp **NGAY SAU** từ gốc `넣다`, từ đơn giản → phức tạp.

## 2. Phân Biệt Từ Đồng Nghĩa (사용하다 vs 이용하다)

```markdown
| 사용하다 | sử dụng | sử dụng | Động từ | **사용**: Sử dụng.<br>**사 (Sử)**: dùng<br>**용 (Dụng)**: công cụ, việc làm<br>→ Nghĩa: **Vận hành/thao tác một vật cụ thể** (máy móc, đồ vật, công cụ).<br><br>**Phân biệt:** Dùng cho **đồ vật hữu hình** mà bạn cầm/chạm vào (điện thoại, máy tính, bút). | 컴퓨터를 **사용해요**. | Tôi **sử dụng** máy tính. | 휴대폰 사용 | 이용하다 | |
| 휴대폰을 사용하다 | | sử dụng điện thoại | Biểu hiện | **휴대폰**: Điện thoại di động. **사용하다**: Sử dụng.<br><br>Cấu trúc: **N을/를 사용하다**.<br>Cấu trúc này có nghĩa là: **Vận hành/thao tác đối tượng N.**<br><br>Trong câu **휴대폰을 사용하다** có nghĩa là: **Sử dụng điện thoại.** | 수업 중에 **휴대폰을 사용하지 마세요**. | Đừng **sử dụng điện thoại** trong giờ học. | | | |
| 이용하다 | lợi dụng | sử dụng, tận dụng | Động từ | **이용**: Lợi dụng.<br>**이 (Lợi)**: có ích<br>**용 (Dụng)**: dùng<br>→ Nghĩa: **Tận dụng một hệ thống/dịch vụ/tiện ích có sẵn** để đạt mục đích.<br><br>**Phân biệt:** Dùng cho **dịch vụ/hệ thống** (phương tiện công cộng, internet, thư viện). | 버스를 **이용해요**. | Tôi **sử dụng (đi)** xe buýt. | 대중교통 이용 | 사용하다 | |
| 버스를 이용하다 | | sử dụng xe buýt | Biểu hiện | **버스**: Xe buýt. **이용하다**: Sử dụng/tận dụng.<br><br>Cấu trúc: **N을/를 이용하다**.<br>Cấu trúc này có nghĩa là: **Tận dụng dịch vụ/phương tiện N.**<br><br>Trong câu **버스를 이용하다** có nghĩa là: **Sử dụng xe buýt (như phương tiện giao thông công cộng).** | 학교에 갈 때 **버스를 이용해요**. | Khi đi học tôi **sử dụng xe buýt**. | | | |
| 도서관을 이용하다 | | sử dụng thư viện | Biểu hiện | **도서관**: Thư viện. **이용하다**: Sử dụng/tận dụng.<br><br>Cấu trúc: **N을/를 이용하다**.<br>Cấu trúc này có nghĩa là: **Tận dụng tiện ích/dịch vụ N.**<br><br>Trong câu **도서관을 이용하다** có nghĩa là: **Sử dụng thư viện (như một tiện ích công cộng).** | 공부할 때 **도서관을 이용해요**. | Khi học tôi **sử dụng thư viện**. | | | |
```

**Note:** Phần "**Phân biệt:**" giúp người học không bị nhầm 2 từ.

## 3. Lỗi Thường Mắc & Cách Sửa

### Tiếng Việt lẫn vào cột Tiếng Hàn

| SAI ❌ | ĐÚNG ✅ |
|--------|---------|
| `가방에 túi을 넣다` | `가방에 핸드폰을 넣다` |
| `부모님은 con을 걱정하세요` | `부모님은 자식을 걱정하세요` |
| `cà phê에 설탕을 chovào어요` | `커피에 설탕을 넣었어요` |
| `hai người은 외모가 비슷해요` | `두 사람은 외모가 비슷해요` |

### Biểu hiện đặt sai vị trí

| SAI ❌ | ĐÚNG ✅ |
|--------|---------|
| Tất cả biểu hiện mới ở cuối file | Biểu hiện ngay sau từ gốc |
