# Templates Nhanh - Copy & Paste

## 1. Template Từ Hán Việt

```markdown
| [TỪ] | [hán việt] | [nghĩa VN] | Động từ | **[TỪ]**: [Hán Việt].<br>**[chữ 1] ([Hán 1])**: [nghĩa]<br>**[chữ 2] ([Hán 2])**: [nghĩa]<br>→ Nghĩa: [tổng hợp]. | [ví dụ tiếng Hàn] | [nghĩa ví dụ] | [collocation] | [synonym] | [antonym] |
```

**Ví dụ:**
```markdown
| 도착하다 | đáo trước | đến nơi | Động từ | **도착**: Đáo trước.<br>**도 (Đáo)**: đến<br>**착 (Trước)**: dính vào/tới nơi<br>→ Nghĩa: Đến nơi, tới đích. | 공항에 **도착했어요**. | Tôi đã **đến** sân bay. | 정시에 도착하다 | | 출발하다 |
```

## 2. Template Từ Thuần Hàn

```markdown
| [TỪ] | | [nghĩa VN] | Động từ | Động từ thuần Hàn.<br>Nghĩa: [giải thích]. | [ví dụ tiếng Hàn] | [nghĩa ví dụ] | | | [antonym] |
```

**Ví dụ:**
```markdown
| 넣다 | | bỏ vào, cho vào | Động từ | Động từ thuần Hàn.<br>Nghĩa: Hành động đặt vật phẩm vào bên trong một không gian nào đó. | 가방에 핸드폰을 **넣었어요**. | Tôi đã **cho** điện thoại vào túi. | | | 꺼내다 |
```

## 3. Template Biểu Hiện (3 Bước)

```markdown
| [BIỂU HIỆN] | | [nghĩa VN] | Biểu hiện | **[từ chính]**: [nghĩa].<br><br>Cấu trúc: **[CẤU TRÚC]**.<br>Cấu trúc này có nghĩa là: **[nghĩa cấu trúc].**<br><br>Trong câu **[biểu hiện]** có nghĩa là: **[nghĩa cụ thể].** | [ví dụ tiếng Hàn] | [nghĩa ví dụ] | | | [antonym nếu có] |
```

**Ví dụ:**
```markdown
| 가방에 넣다 | | bỏ vào túi | Biểu hiện | **가방**: Túi. **넣다**: Bỏ vào.<br><br>Cấu trúc: **N1에 N2을/를 넣다**.<br>Cấu trúc này có nghĩa là: **Bỏ vật phẩm N2 vào không gian N1.**<br><br>Trong câu **가방에 넣다** có nghĩa là: **Bỏ (cái gì đó) vào túi.** | 책을 **가방에 넣었어요**. | Tôi đã **bỏ sách vào túi**. | | | 가방에서 꺼내다 |
```

## 4. Template Từ Đồng Nghĩa (Có Phân Biệt)

```markdown
| [TỪ 1] | [hán việt] | [nghĩa] | Động từ | **[TỪ]**: [Hán Việt].<br>→ Nghĩa: [nghĩa chính].<br><br>**Phân biệt:** Dùng cho **[bối cảnh cụ thể]**. | [ví dụ] | [nghĩa ví dụ] | | [từ đồng nghĩa] | |
```

**Ví dụ:**
```markdown
| 사용하다 | sử dụng | sử dụng | Động từ | **사용**: Sử dụng.<br>**사 (Sử)**: dùng<br>**용 (Dụng)**: công cụ<br>→ Nghĩa: **Vận hành/thao tác một vật cụ thể**.<br><br>**Phân biệt:** Dùng cho **đồ vật hữu hình** (điện thoại, máy tính). | 컴퓨터를 **사용해요**. | Tôi **sử dụng** máy tính. | | 이용하다 | |
```

## 5. Template Từ Mượn (Loanword)

```markdown
| [TỪ] | [gốc] | [nghĩa VN] | Danh từ | Từ mượn tiếng [Anh/Pháp] "[gốc]". | [ví dụ] | [nghĩa ví dụ] | | | |
```

**Ví dụ:**
```markdown
| 버스 | bus | xe buýt | Danh từ | Từ mượn tiếng Anh "bus". | **버스**를 타요. | Tôi lên **xe buýt**. | | | |
| 크림 | cream | kem | Danh từ | Từ mượn tiếng Anh "cream".<br>Nghĩa: Loại chất lỏng sệt dùng trong thực phẩm hoặc mỹ phẩm. | 케이크에 **크림**이 많아요. | Trong bánh có nhiều **kem**. | | | |
```
