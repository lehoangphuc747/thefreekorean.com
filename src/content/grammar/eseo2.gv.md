## 1. Khởi động: Xuất phát từ đâu? 🛫

**Tư duy ngữ pháp:**
> **[Địa điểm/Nguồn] + 에서 + [Động từ]**

* **Nghĩa tiếng Việt:** Từ... (chỉ điểm bắt đầu).
* **Trọng tâm:** Diễn tả nơi hành động bắt đầu, hoặc nguồn gốc tin tức/vật chất.
* **Ví dụ:** *Đến từ* Hàn Quốc, *Đi từ* nhà, *Nhận tin từ* trường.

---

## 2. Quy tắc kết hợp ⚡️

**Quy tắc:** Luôn gắn **에서** vào sau danh từ (Không phân biệt Patchim).

| Danh từ | + 에서 | Kết quả |
| :--- | :---: | :--- |
| **서울** (Seoul) | + 에서 | **서울에서** |
| **학교** (Trường) | + 에서 | **학교에서** |
| **집** (Nhà) | + 에서 | **집에서** |
| **중국** (Trung Quốc) | + 에서 | **중국에서** |

## 3. Chủ đề 1: Địa điểm xuất phát 🗺️

**Bước 1: Luyện cụm từ (Nơi bắt đầu)**
* Từ Seoul: **서울에서**
* Từ nhà: **집에서**
* Từ sân bay: **공항에서**

    ➡ ........................................................................

    ➡ ........................................................................

    ➡ ........................................................................

## 5. So sánh quan trọng ⚖️

### A. **에서** vs **부터** (Từ...)

| Đặc điểm | **에서** | **부터** |
| :--- | :--- | :--- |
| **Đối tượng** | Thiên về **Địa điểm** | Thiên về **Thời gian** |
| **Ví dụ đúng** | **서울에서** 출발해요. | **9시부터** 시작해요. |
| **Ví dụ sai ❌** | 9시**에서** 시작해요. | |

### B. **에서** vs **에게서/한테서** (Từ ai?)

| Đặc điểm | **에서** | **에게서 / 한테서** |
| :--- | :--- | :--- |
| **Đối tượng** | **Địa điểm, Tổ chức** (Trường, Công ty...) | **Con người, Động vật** (Bạn bè, Mẹ...) |
| **Ví dụ** | **회사에서** 전화가 왔어요.<br>*(Từ công ty)* | **친구에게서** 전화가 왔어요.<br>*(Từ bạn)* |

## 6. Bài tập thực hành ✍️

**Bài 1: Dịch phản xạ (Việt ➡ Hàn)**

1.  Từ Seoul ➡ ........................
2.  Từ 9 giờ (Thời gian - chú ý!) ➡ ........................
3.  Từ trường học ➡ ........................
4.  Từ người bạn (Con người - chú ý!) ➡ ........................

**Bài 2: Sắp xếp câu hoàn chỉnh**

1.  (출발해요 / 내일 / 서울 / 에서)
    ➡ ........................................................................

2.  (얼마나 / 학교 / 에서 / 걸려요 / ?)
    ➡ ........................................................................

**Bài 3: Thử thách điền từ (Điền cả cụm)**

1.  Tôi đến **(từ Việt Nam)**.
    ➡ 저는 ........................................ 왔어요.

2.  Điện thoại gọi đến **(từ nhà)**.
    ➡ ........................................ 전화가 왔어요.
