## 1. 단위명사 — Danh từ đơn vị

| 한국어 | Nghĩa | Loại từ | Giải thích | Ví dụ | Nghĩa ví dụ |
|--------|-------|--------|----------|-------|-------------|
| 벌 | bộ (quần áo) | Danh từ đơn vị |  | 옷 한 벌, 바지 한 벌 | một bộ quần áo, một bộ quần |
| 옷 한 벌 | một bộ quần áo | Biểu hiện |  | 옷 한 벌을 사다 | mua một bộ quần áo |
| 송이 | đóa, chùm | Danh từ đơn vị |  | 꽃 한 송이, 포도 한 송이 | một đóa hoa, một chùm nho |
| 꽃 한 송이 | một đóa hoa | Biểu hiện |  | 꽃 한 송이를 사다 | mua một đóa hoa |
| 켤레 | đôi (giày, tất) | Danh từ đơn vị | Dùng cho giày/tất (cặp trái-phải) — khác với 쌍 | 구두 한 켤레, 양말 한 켤레 | một đôi giày da, một đôi tất |
| 구두 한 켤레 | một đôi giày da | Biểu hiện |  | 구두 한 켤레를 사다 | mua một đôi giày da |
| 바구니 | giỏ, rổ | Danh từ đơn vị |  | 귤 한 바구니, 과일 한 바구니 | một giỏ quýt, một giỏ trái cây |
| 귤 한 바구니 | một giỏ quýt | Biểu hiện |  | 귤 한 바구니를 사다 | mua một giỏ quýt |
| 쌍 | đôi (khuyên tai) | Danh từ đơn vị | Dùng cho đồ vật thành cặp như khuyên tai, đũa — khác 켤레 | 귀걸이 한 쌍, 젓가락 한 쌍 | một đôi khuyên tai, một đôi đũa |
| 귀걸이 한 쌍 | một đôi khuyên tai | Biểu hiện |  | 귀걸이 한 쌍을 사다 | mua một đôi khuyên tai |
| 상자 | hộp, thùng | Danh từ đơn vị |  | 귤 한 상자, 사과 한 상자 | một thùng quýt, một thùng táo |
| 귤 한 상자 | một thùng quýt | Biểu hiện |  | 귤 한 상자를 가져오다 | mang một thùng quýt |
| 봉지 | túi ni lông | Danh từ đơn vị |  | 귤 한 봉지, 과자 한 봉지 | một túi quýt, một túi bánh |
| 귤 한 봉지 | một túi quýt | Biểu hiện |  | 귤 한 봉지를 사다 | mua một túi quýt |
| 장 | tờ, trang | Danh từ đơn vị |  | 종이 한 장, 사진 한 장 | một tờ giấy, một tấm ảnh |
| 종이 한 장 | một tờ giấy | Biểu hiện |  | 종이 한 장을 주다 | đưa một tờ giấy |

## 3. 구매/탐색 — Mua sắm & tìm kiếm

| 한국어 | Nghĩa | Loại từ | Giải thích | Ví dụ | Nghĩa ví dụ |
|--------|-------|--------|----------|-------|-------------|
| 찾다 | tìm kiếm | Động từ |  | 뭘 찾으세요? | Bạn đang tìm gì vậy? |
| 주문하다 | đặt hàng | Động từ |  | 온라인으로 주문하다 | đặt hàng online |
| 팔다 | bán | Động từ |  | 이 가게에서 과일을 팔다 | cửa hàng này bán trái cây |
| 과일을 팔다 | bán trái cây | Biểu hiện |  | 시장에서 과일을 팔다 | bán trái cây ở chợ |
| 필요하다 | cần thiết | Động từ |  | 지금 필요하다 | hiện tại cần |
| 우산이 필요하다 | cần ô | Biểu hiện |  | 비가 오면 우산이 필요하다 | trời mưa thì cần ô |
| 유행하다 | thịnh hành, đang mốt | Động từ |  | 요즘 유행하다 | đang thịnh hành dạo này |
| 이 디자인이 유행하다 | thiết kế này đang thịnh hành | Biểu hiện |  | 요즘 이 디자인이 유행하다 | dạo này thiết kế này đang thịnh hành |
| 제품 | sản phẩm | Danh từ |  | 제품이 다양하다 | sản phẩm đa dạng |
| 다양하다 | đa dạng | Tính từ |  | 제품이 다양하다 | sản phẩm đa dạng |
| 제품이 다양하다 | sản phẩm đa dạng | Biểu hiện |  | 이 가게는 제품이 다양하다 | cửa hàng này sản phẩm đa dạng |
| 싸다 | rẻ | Tính từ |  | 할인점에서 싸게 사다 | mua rẻ ở cửa hàng giảm giá |
| 빠르다 | nhanh | Tính từ |  | 배달이 빠르다 | giao hàng nhanh |
| 배달이 빠르다 | giao hàng nhanh | Biểu hiện |  | 이 쇼핑몰은 배달이 빠르다 | trung tâm thương mại này giao hàng nhanh |
| 쉽다 | dễ | Tính từ |  | 사용하기 쉽다 | dễ sử dụng |
| 쉽게 | một cách dễ dàng | Trạng từ | Dạng trạng từ của 쉽다; tính từ + -게 = trạng từ | 쉽게 살 수 있다 | có thể mua dễ dàng |
| 쉽게 사다 | mua (một cách) dễ | Biểu hiện |  | 온라인으로 쉽게 사다 | mua dễ dàng qua mạng |
| 누구나 | bất kỳ ai | Đại từ | 누구 (ai) + 나 (nhấn mạnh) → bất kỳ ai, ai cũng | 누구나 살 수 있다 | bất kỳ ai cũng có thể mua |
| 누구나 살 수 있다 | bất kỳ ai cũng có thể mua | Biểu hiện |  | 이 앱은 누구나 살 수 있다 | app này ai cũng có thể mua |
| 이것만 | chỉ cái này | Đại từ + trợ từ | 만 = chỉ/mà thôi (tiểu từ giới hạn, gắn sau danh từ) | 이것만 주세요 | chỉ cho tôi cái này thôi |
| 책만 사다 | chỉ mua sách | Biểu hiện | 만 gắn vào danh từ → "chỉ N thôi": 책만 = chỉ sách | 오늘은 책만 사다 | hôm nay chỉ mua sách |
| 기분 | tâm trạng, cảm giác | Danh từ | Tâm trạng tại thời điểm cụ thể — khác 감정 (cảm xúc nói chung) | 기분이 좋다 | tâm trạng tốt |
| 기분이 좋다 | tâm trạng tốt, vui | Biểu hiện |  | 쇼핑하면 기분이 좋다 | mua sắm thì tâm trạng vui |
| 기분 좋은 가격 | giá cả phải chăng | Biểu hiện | 기분 좋은 = tính từ ghép: "mang lại tâm trạng tốt" → phải chăng | 이 가게는 기분 좋은 가격이에요 | cửa hàng này có giá cả phải chăng |

## 4. 계산/결제 — Thanh toán

| 한국어 | Nghĩa | Loại từ | Giải thích | Ví dụ | Nghĩa ví dụ |
|--------|-------|--------|----------|-------|-------------|
| 카드 | thẻ (card) | Danh từ |  | 카드로 계산하다 | thanh toán bằng thẻ |
| 계산하다 | tính tiền, thanh toán | Động từ |  | 어떻게 계산하시겠어요? | Bạn muốn thanh toán thế nào? |
| 카드로 계산하다 | thanh toán bằng thẻ | Biểu hiện |  | 카드로 계산하겠습니다 | tôi sẽ thanh toán bằng thẻ |
| 교환하다 | đổi đồ (sang đồ khác) | Động từ | Đổi sang đồ khác, không lấy tiền — khác 환불하다 | 사이즈를 교환하다 | đổi size |
| 환불하다 | trả đồ (lấy lại tiền) | Động từ | Trả đồ để lấy lại tiền — khác 교환하다 | 물건을 환불하다 | trả hàng lấy tiền |
| 카드도 돼요? | Thẻ cũng được không? | Biểu hiện |  | 현금 없는데 카드도 돼요? | Tôi không có tiền mặt, thẻ được không? |
| 카드로 하겠습니다. | Tôi sẽ thanh toán bằng thẻ. | Biểu hiện |  | 현금 말고 카드로 하겠습니다. | Không dùng tiền mặt, tôi sẽ trả bằng thẻ. |
| 어떻게 계산하시겠어요? | Bạn muốn thanh toán thế nào? | Biểu hiện |  | 현금이요, 카드요? 어떻게 계산하시겠어요? | Tiền mặt hay thẻ? Bạn muốn thanh toán thế nào? |

## 5. 옷/패션 — Quần áo & thời trang

| 한국어 | Nghĩa | Loại từ | Giải thích | Ví dụ | Nghĩa ví dụ |

| 디자인 | thiết kế | Danh từ |  | 디자인이 예쁘다 | thiết kế đẹp |
| 화려하다 | sặc sỡ, lộng lẫy | Tính từ | Lộng lẫy, phô trương — mạnh hơn "nhiều màu sắc" | 치마가 화려하다 | váy sặc sỡ |
| 치마가 화려하다 | váy sặc sỡ | Biểu hiện |  | 이 치마가 화려하다 | cái váy này sặc sỡ |
| 단순하다 | đơn giản | Tính từ |  | 디자인이 단순하다 | thiết kế đơn giản |
| 디자인이 단순하다 | thiết kế đơn giản | Biểu hiện |  | 이 옷은 디자인이 단순하다 | cái áo này thiết kế đơn giản |
| 길다 | dài | Tính từ |  | 치마가 길다 | váy dài |
| 치마가 길다 | váy dài | Biểu hiện |  | 이 치마가 길다 | cái váy này dài |
| 짧다 | ngắn | Tính từ |  | 소매가 짧다 | tay áo ngắn |
| 어울리다 | phù hợp, hợp với | Động từ | Hợp với người/phong cách; khác 맞다 (vừa về kích cỡ) | 잘 어울려요 | Rất hợp với bạn |
| 마음에 들다 | vừa ý, thích | Động từ | 마음 = lòng → "lọt vào lòng" → cảm thấy thích/vừa ý | 이 옷이 마음에 들어요 | Tôi thích cái áo này |
| 맞다 | vừa (kích cỡ) | Động từ | Nhiều nghĩa; ở đây = vừa size, không phải "đúng/phải" | 사이즈가 맞다 | vừa size |
| 사이즈가 맞다 | size vừa | Biểu hiện |  | 이 신발 사이즈가 맞다 | đôi giày này vừa size |
| 예쁜 디자인 | thiết kế đẹp | Biểu hiện |  | 이 가방은 예쁜 디자인이에요 | cái túi này có thiết kế đẹp |
| 이 옷이 어때요? | Bộ quần áo này thế nào? | Biểu hiện |  | 이 옷이 어때요? 잘 어울려요? | Bộ này thế nào? Có hợp không? |
| 잘 어울려요. | Rất hợp với bạn. | Biểu hiện |  | 그 색깔이 잘 어울려요. | Màu đó rất hợp với bạn. |
