# Hướng dẫn đăng tài liệu

Bạn có thể viết bài tài liệu dưới dạng file `.mdx` trong thư mục `src/documents/`.

## Cấu trúc frontmatter (phần đầu file)

```mdx
---
title: "Tên tài liệu"
date: YYYY-MM-DD
category: "Tên danh mục chính" # Ví dụ: Giáo trình, Tài liệu, Video, Trang web
subcategory: "Tên danh mục con" # Ví dụ: Tiếng Hàn Tổng Hợp, Seoul, ...
type: "Loại file" # PDF, Word, Video, Website...
cover: "/images/ten-anh-bia.jpg" # Đường dẫn ảnh bìa (nếu có)
---
```

## Nội dung bài viết

Bạn có thể sử dụng markdown để trình bày:

- **Tiêu đề:** Dùng `#`, `##`, ...
- **Chèn ảnh:**
  ```mdx
  ![Tên ảnh](đường-dẫn-ảnh)
  ```
- **Chèn link:**
  ```mdx
  [Tên link](https://duong-dan.com)
  ```
- **Danh sách:**
  ```mdx
  - Mục 1
  - Mục 2
  ```
