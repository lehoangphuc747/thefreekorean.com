# Ngữ pháp: -어서/-아서 (vì, nên)

## 📝 Ý nghĩa
Cấu trúc **-어서/-아서** diễn tả quan hệ nguyên nhân - kết quả giữa hai hành động hoặc trạng thái. Mệnh đề đầu là nguyên nhân, mệnh đề sau là kết quả.

## 📚 Cấu trúc ngữ pháp

### Quy tắc âm vực học (Vowel Harmony)
```
• Batchy âm cuối có ㅏ, ㅗ → -아서
• Batchy âm cuối có ㅓ, ㅜ, ㅡ, ㅣ → -어서  
• 하다 → 해서 (đặc biệt)
```

### Chi tiết biến đổi:
- **ㅏ/ㅗ + 아서**: 가다 → 가서, 좋다 → 좋아서
- **ㅓ/ㅜ/ㅡ/ㅣ + 어서**: 먹다 → 먹어서, 크다 → 커서
- **하다 → 해서**: 공부하다 → 공부해서

## 🎯 Các cách sử dụng

### 1. Nguyên nhân - Kết quả tự nhiên
```korean
비가 와서 집에 있어요.
[bi-ga wa-seo ji-be i-sseo-yo]
Trời mưa nên ở nhà.
```

### 2. Hành động liên tiếp có mối liên hệ
```korean
일찍 일어나서 운동을 해요.
[il-jjik i-reo-na-seo un-dong-eul hae-yo]
Dậy sớm nên tập thể dục. / Dậy sớm để tập thể dục.
```

### 3. Lý do cho trạng thái hiện tại
```korean
많이 먹어서 배불러요.
[ma-ni meo-geo-seo bae-bul-leo-yo]
Ăn nhiều nên no.
```

## 🔍 Phân loại cách dùng

### 1. Nguyên nhân khách quan
```korean
비가 와서 소풍을 못 갔어요.
Trời mưa nên không thể đi dã ngoại.
```

### 2. Hành động có mục đích
```korean
돈을 벌어서 여행을 가고 싶어요.
Kiếm tiền để đi du lịch.
```

### 3. Trạng thái dẫn đến kết quả
```korean
너무 피곤해서 일찍 잤어요.
Quá mệt nên ngủ sớm.
```

## 🔄 So sánh với các cấu trúc khác

| Cấu trúc | Đặc điểm | Ví dụ |
|----------|----------|-------|
| **-어서/-아서** | Nguyên nhân tự nhiên | 아파서 못 갔어요 (đau nên không đi được) |
| **-니까** | Lý do chủ quan | 아프니까 집에 가세요 (đau nên về nhà đi) |
| **-기 때문에** | Nguyên nhân chính thức | 비 때문에 못 갔어요 (vì mưa nên không đi được) |

## ⚠️ Lưu ý quan trọng

### 1. Không dùng với lệnh/đề nghị
❌ **Sai**: 와서 같이 가요. (Come and let's go together)
✅ **Đúng**: 와서 같이 갔어요. (Came and went together)

### 2. Mệnh đề sau phải là kết quả tự nhiên
✅ 배가 고파서 밥을 먹었어요. (Đói nên ăn cơm)
✅ 돈이 없어서 못 샀어요. (Không có tiền nên không mua được)

### 3. Biến đổi bất quy tắc
- **ㅂ bất quy tắc**: 덥다 → 더워서, 춥다 → 추워서
- **ㄷ bất quy tắc**: 걷다 → 걸어서, 듣다 → 들어서
- **ㅅ bất quy tắc**: 낫다 → 나아서

## 📖 Ví dụ mở rộng

1. **Thời tiết**: 눈이 와서 길이 미끄러워요.
2. **Cảm xúc**: 슬퍼서 울었어요.
3. **Sức khỏe**: 감기에 걸려서 병원에 갔어요.
4. **Tình huống**: 늦어서 뛰어갔어요.
