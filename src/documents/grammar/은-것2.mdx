# Ngữ pháp: -은 것2 (Tính từ + 은 것)

## 📝 Ý nghĩa
Cấu trúc **-은 것** với tính từ được sử dụng để danh từ hóa tính từ, biến một tính từ thành một danh từ để chỉ "cái gì đó có tính chất/đặc điểm nào đó".

## 📚 Cấu trúc ngữ pháp

### Với tính từ có batchy cuối (있다/없다 exception)
```
형용사 어간 + ㄴ/은 것
```

### Với tính từ khác
```
형용사 어간 + ㄴ 것
```

### Chi tiết cấu tạo:
- **Tính từ có batchy cuối kết thúc bằng consonant**: + 은 것
  - 작다 → 작은 것 (cái nhỏ)
  - 좋다 → 좋은 것 (cái tốt)
- **Tính từ có batchy cuối kết thúc bằng vowel**: + ㄴ 것  
  - 예쁘다 → 예쁜 것 (cái đẹp)
  - 크다 → 큰 것 (cái to)

## 🔄 So sánh với -은 것1 (Động từ)

| | -은 것1 (Động từ) | -은 것2 (Tính từ) |
|---|---|---|
| **Ý nghĩa** | Hành động đã xảy ra | Đặc điểm, tính chất |
| **Ví dụ** | 먹은 것 (cái đã ăn) | 맛있는 것 (cái ngon) |
| **Thời gian** | Quá khứ | Hiện tại/Tổng quát |

## 🎯 Cách sử dụng chính

### 1. Chỉ đối tượng có đặc điểm cụ thể
```korean
좋은 것을 고르세요.
[jo-eun geo-seul go-reu-se-yo]
Hãy chọn cái tốt.
```

### 2. So sánh giữa các đối tượng
```korean
새로운 것이 오래된 것보다 좋아요.
[sae-ro-un geo-si o-rae-dwen geot-bo-da jo-a-yo]
Cái mới tốt hơn cái cũ.
```

### 3. Mô tả sở thích
```korean
달콤한 것을 좋아해요.
[dal-kom-han geo-seul jo-a-hae-yo]
Tôi thích những cái ngọt.
```

## ⚠️ Lưu ý quan trọng

### 1. Phân biệt với động từ
- **Tính từ**: 좋은 것 (cái tốt - tính chất)
- **Động từ**: 좋아한 것 (cái đã thích - hành động)

### 2. Biến đổi theo thời gian (ít dùng)
- **Hiện tại**: 좋은 것 (thường dùng)
- **Quá khứ**: 좋았던 것 (cái đã từng tốt)
- **Tương lai**: 좋을 것 (cái sẽ tốt)

### 3. Kết hợp với các trợ từ
```korean
좋은 것을/을 - cái tốt (tân ngữ)
좋은 것이/가 - cái tốt (chủ ngữ) 
좋은 것으로 - bằng/với cái tốt
```

## 📖 Ví dụ mở rộng

1. **Mua sắm**: 싼 것만 사요. (Chỉ mua cái rẻ thôi.)
2. **Ẩm thực**: 매운 것을 못 먹어요. (Tôi không ăn được cái cay.)
3. **Sở thích**: 재미있는 것을 좋아해요. (Tôi thích cái thú vị.)
4. **So sánh**: 새것이 헌 것보다 좋아요. (Cái mới tốt hơn cái cũ.)
