# Ngữ pháp: -니까 (vì, bởi vì)

## 📝 Ý nghĩa
Cấu trúc **-니까** diễn tả lý do hoặc nguyên nhân mang tính chủ quan. Thường được dùng khi người nói đưa ra lời khuyên, yêu cầu, đề nghị dựa trên lý do đó.

## 📚 Cấu trúc ngữ pháp

### Với động từ/tính từ
```
• Batchy âm cuối kết thúc bằng vowel: 어간 + 니까
• Batchy âm cuối kết thúc bằng consonant: 어간 + 으니까
```

### Chi tiết biến đổi:
- **Vowel ending**: 가다 → 가니까, 크다 → 크니까
- **Consonant ending**: 먹다 → 먹으니까, 좋다 → 좋으니까
- **ㄹ 불규칙**: 울다 → 우니까 (ㄹ drops)
- **하다**: 하다 → 하니까

## 🎯 Đặc điểm sử dụng

### 1. Lời khuyên dựa trên lý do
```korean
비가 오니까 우산을 가져가세요.
[bi-ga o-ni-kka u-sa-neul ga-jyeo-ga-se-yo]
Trời mưa nên hãy mang theo ô.
```

### 2. Yêu cầu với lý do
```korean
시끄러우니까 조용히 하세요.
[si-kkeu-reo-u-ni-kka jo-yong-hi ha-se-yo]
Ồn ào quá nên hãy im lặng.
```

### 3. Đề xuất với căn cứ
```korean
맛있으니까 드셔 보세요.
[ma-si-ssu-ni-kka deu-syeo bo-se-yo]
Ngon nên hãy thử ăn xem.
```

## 🔍 Ngữ cảnh sử dụng

### 1. Trong hội thoại (đặc biệt quan trọng)
```korean
A: 왜 안 가요? (Tại sao không đi?)
B: 바쁘니까요. (Vì bận.)
```

### 2. Lời khuyên từ người lớn tuổi hơn
```korean
공부를 열심히 하니까 성공할 거예요.
Học chăm chỉ nên sẽ thành công.
```

### 3. Giải thích quyết định
```korean
비싸니까 다른 걸 사겠어요.
Đắt nên tôi sẽ mua cái khác.
```

## 🔄 So sánh với -어서/-아서

| Đặc điểm | -어서/-아서 | -니까 |
|----------|-------------|--------|
| **Tính chất** | Nguyên nhân khách quan | Lý do chủ quan |
| **Sử dụng** | Kết quả tự nhiên | Lời khuyên, yêu cầu |
| **Ví dụ** | 아파서 못 갔어요 | 아프니까 가지 마세요 |
| **Ngữ cảnh** | Giải thích sự việc | Đưa ra đề xuất |

## ⚠️ Lưu ý quan trọng

### 1. Thường dùng với lời khuyên
✅ 위험하니까 조심하세요. (Nguy hiểm nên cẩn thận)
✅ 맛있으니까 드셔 보세요. (Ngon nên thử ăn xem)

### 2. Có thể dùng trong câu hỏi
```korean
왜 안 가니까요? (Tại sao không đi vậy?)
```

### 3. Dùng khi giải thích quyết định của mình
```korean
피곤하니까 일찍 잘 거예요.
Mệt nên tôi sẽ ngủ sớm.
```

### 4. Biến đổi bất quy tắc
- **ㄹ 불규칙**: 울다 → 우니까, 살다 → 사니까
- **ㅂ 불규칙**: 덥다 → 더우니까, 춥다 → 추우니까
- **ㄷ 불규칙**: 걷다 → 걸으니까, 듣다 → 들으니까

## 📖 Ví dụ thực tế

1. **Lời khuyên**: 아프니까 병원에 가세요.
2. **Yêu cầu**: 시끄러우니까 조용히 해 주세요.
3. **Đề xuất**: 재미있으니까 같이 봐요.
4. **Giải thích**: 바쁘니까 내일 만나요.
