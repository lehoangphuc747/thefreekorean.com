# Ngữ pháp: -지만 (nhưng, mặc dù)

## 📝 Ý nghĩa
Cấu trúc **-지만** diễn tả sự đối lập hoặc tương phản giữa hai mệnh đề. Nó tương đương với "nhưng", "mặc dù", "tuy nhiên" trong tiếng Việt.

## 📚 Cấu trúc ngữ pháp

### Cấu trúc cơ bản
```
동사/형용사/명사 어간 + 지만
```

### Cách tạo thành:
- **Động từ/Tính từ**: 어간 + 지만
  - 가다 → 가지만 (đi nhưng)
  - 좋다 → 좋지만 (tốt nhưng)
- **Danh từ + 이다**: 이지만/지만
  - 학생이지만 (là học sinh nhưng)
  - 의사지만 (là bác sĩ nhưng)

## 🎯 Các cách sử dụng

### 1. Đối lập đơn giản
```korean
비싸지만 좋아요.
[bi-ssa-ji-man jo-a-yo]
Đắt nhưng tốt.
```

### 2. Mặc dù... nhưng...
```korean
어렵지만 재미있어요.
[eo-ryeop-ji-man jae-mi-is-seo-yo]
Khó nhưng thú vị.
```

### 3. Thừa nhận một điều, phủ nhận điều khác
```korean
한국어를 배우지만 아직 잘 못해요.
[han-gug-eo-reul bae-u-ji-man a-jik jal mo-tae-yo]
Đang học tiếng Hàn nhưng chưa giỏi.
```

## 🔍 Các tình huống sử dụng

### 1. Mô tả tính cách/sự vật có mặt trái
```korean
조용하지만 재미있는 사람이에요.
Người im lặng nhưng thú vị.
```

### 2. Thừa nhận hạn chế nhưng vẫn làm
```korean
어렵지만 열심히 공부할 거예요.
Khó nhưng sẽ học chăm chỉ.
```

### 3. So sánh hai khía cạnh
```korean
가격은 싸지만 품질이 나빠요.
Giá rẻ nhưng chất lượng kém.
```

## 🔄 So sánh với các liên từ khác

| Liên từ | Ý nghĩa | Đặc điểm |
|---------|---------|----------|
| **-지만** | nhưng, mặc dù | Đối lập rõ ràng |
| **-고** | và, rồi | Kết nối tự nhiên |
| **-어서/-아서** | nên, vì | Nguyên nhân - kết quả |
| **-는데/-은데** | mà, nhưng | Đối lập nhẹ nhàng |

## ⚠️ Lưu ý quan trọng

### 1. Luôn giữ nguyên dạng
**-지만** không thay đổi theo thời gian hay ngôi:
- 좋지만 (tốt nhưng) - hiện tại
- 좋았지만 (đã tốt nhưng) - quá khứ  
- 좋겠지만 (có lẽ tốt nhưng) - phỏng đoán

### 2. Có thể dùng ở cuối câu
```korean
A: 어때요? (Thế nào?)
B: 좋지만요. (Tốt nhưng...)
```

### 3. Với danh từ
- **받침 있는 명사**: 명사 + 이지만
- **받침 없는 명사**: 명사 + 지만
```korean
학생이지만 (học sinh nhưng)
의사지만 (bác sĩ nhưng)
```

## 📖 Ví dụ mở rộng

1. **Tính cách**: 조용하지만 친절해요.
2. **Khả năng**: 할 수 있지만 하기 싫어요.
3. **Tình huống**: 바쁘지만 시간을 내겠어요.
4. **Đánh giá**: 저렴하지만 품질이 좋아요.
