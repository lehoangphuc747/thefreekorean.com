# Ngữ pháp: -면 (nếu, khi)

## 📝 Ý nghĩa
Cấu trúc **-면** diễn tả điều kiện giả định ("nếu") hoặc thời gian trong tương lai ("khi"). Đây là một trong những cấu trúc điều kiện cơ bản và quan trọng nhất trong tiếng Hàn.

## 📚 Cấu trúc ngữ pháp

### Với động từ/tính từ
```
• Batchy âm cuối kết thúc bằng vowel: 어간 + 면
• Batchy âm cuối kết thúc bằng consonant: 어간 + 으면
```

### Chi tiết biến đổi:
- **Vowel ending**: 가다 → 가면, 크다 → 크면
- **Consonant ending**: 먹다 → 먹으면, 좋다 → 좋으면
- **ㄹ 불규칙**: 살다 → 살면 (ㄹ drops)
- **하다**: 하다 → 하면

## 🎯 Các cách sử dụng

### 1. Điều kiện giả định (nếu)
```korean
비가 오면 집에 있을 거예요.
[bi-ga o-myeon ji-be i-sseul geo-ye-yo]
Nếu trời mưa thì sẽ ở nhà.
```

### 2. Thời gian tương lai (khi)
```korean
한국에 가면 김치를 먹어 보세요.
[han-gu-ge ga-myeon kim-chi-reul meo-geo bo-se-yo]
Khi đến Hàn Quốc, hãy thử ăn kim chi.
```

### 3. Lời khuyên có điều kiện
```korean
돈이 있으면 여행을 하세요.
[do-ni i-sseu-myeon yeo-haeng-eul ha-se-yo]
Nếu có tiền thì hãy đi du lịch.
```

## 🔍 Phân loại cách dùng

### 1. Điều kiện có thể xảy ra
```korean
돈을 벌면 차를 살 거예요.
Nếu kiếm được tiền thì sẽ mua xe.
```

### 2. Điều kiện tổng quát
```korean
물을 끓이면 수증기가 나와요.
Nếu đun sôi nước thì sẽ có hơi nước.
```

### 3. Lời khuyên/đề xuất
```korean
피곤하면 쉬세요.
Nếu mệt thì hãy nghỉ.
```

## 🔄 So sánh với các cấu trúc khác

| Cấu trúc | Ý nghĩa | Đặc điểm |
|----------|---------|----------|
| **-면** | nếu, khi | Điều kiện/thời gian |
| **-니까** | vì, do | Lý do chủ quan |
| **-어서/-아서** | vì, nên | Nguyên nhân khách quan |
| **-는데/-은데** | mà, trong khi | Tình huống/nền tảng |

## ⚠️ Lưu ý quan trọng

### 1. Mệnh đề chính thường ở tương lai
```korean
내일 오면 만나요. (Nếu ngày mai đến thì gặp nhau)
```

### 2. Có thể dùng với thời gian hiện tại tổng quát
```korean
물을 끓이면 뜨거워져요.
Nếu đun nước thì sẽ nóng lên.
```

### 3. Biến đổi bất quy tắc
- **ㄹ 불규칙**: 울다 → 울면, 살다 → 살면
- **ㅂ 불규칙**: 덥다 → 더우면, 춥다 → 추우면  
- **ㄷ 불규칙**: 걷다 → 걸으면, 듣다 → 들으면

### 4. Với 있다/없다
```korean
시간이 있으면... (nếu có thời gian...)
돈이 없으면... (nếu không có tiền...)
```

## 📖 Ví dụ thực tế

1. **Lời hứa**: 집에 가면 전화할게요.
2. **Kế hoạch**: 돈을 모으면 집을 살 거예요.
3. **Lời khuyên**: 아프면 병원에 가세요.
4. **Điều kiện**: 날씨가 좋으면 소풍을 가요.
